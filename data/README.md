# AISHE Data

Place your AISHE university dataset here before running the pipeline.

**Expected file:** `AISHE.xlsx` (or set `AISHE_DATA_PATH` in `.env`)

The Excel file is not included in the repository due to size. Download or obtain the AISHE dataset and copy it into this folder.

```powershell
# Example .env setting
AISHE_DATA_PATH=data/AISHE.xlsx
```
